# Free-to-Play Game Explorer

## Project Overview
Free-to-Play Game Explorer is a web application that allows users to discover and explore free-to-play games. The application fetches real-time game data from a public API and presents it through a responsive and user-friendly interface.

## Problem Statement
Many teenage gamers, especially those with limited financial resources, struggle to find quality free-to-play games. Information about such games is often scattered across multiple platforms, making the discovery process inefficient and time-consuming. This project aims to solve this problem by providing a centralized platform where users can easily search, filter, and explore free-to-play games.

## Purpose
The objective of this project is to demonstrate the understanding of JavaScript concepts such as API integration, dynamic data rendering, the use of array higher-order functions, and responsive UI development.

## API Used
FreeToGame Public API  
https://www.freetogame.com/api-doc

This API provides information about free-to-play games including title, genre, platform, release date, description, and thumbnails.

## Planned Features
- Search games by title
- Filter games based on genre or platform
- Sort games by release date or popularity
- Dark mode and light mode toggle
- Responsive design across devices
- Add games to favourites using local storage
